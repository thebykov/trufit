== Description ==

The Easy Timeout Session WordPress plugin allows you to change the session
duration for the WordPress user.

1. Open Timeout Session Page

2. Specify the session length (you can specify in seconds, hours or days)

Do you use worpdress admin with an iPad? DonÂ´t worry, this plugin is fully

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload plugin folder `easy-timeout-session` to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= I cannot set the timeout lower than 5 minutes =

By security, the minimum is 5 minutes (300 seconds). If you want to set a lower timeout, send me an email.
